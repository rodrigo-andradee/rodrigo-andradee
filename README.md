## 📂 Projects

### 🔹 [Insurance ETL Pipeline + Streamlit App](https://github.com/rodrigo-andradee/insurance-etl-pipeline)
End-to-end ETL pipeline on insurance data using **Medallion Architecture** (Bronze/Silver/Gold) in Databricks, with PySpark transformations and Star Schema modelling.  
Includes an interactive lookup app deployed on **Streamlit Cloud** → [insurance-lookup-tool.streamlit.app](https://insurance-lookup-tool.streamlit.app)

### 🔹 [F1 ETL Pipeline](https://github.com/rodrigo-andradee/f1-etl-pipeline)
Ingestion of Formula 1 data via **OpenF1 API** and historical weather data via **Open-Meteo API**, processed with PySpark using Medallion Architecture and Star Schema modelling in Databricks.

### 🔹 [Olist ETL Pipeline](https://github.com/rodrigo-andradee/olist-etl-pipeline)
ETL pipeline on the public Brazilian e-commerce dataset **Olist**, built with PySpark across five layer-separated notebooks following Medallion Architecture and Star Schema modelling.
